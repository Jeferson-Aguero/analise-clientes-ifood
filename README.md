# analise-clientes-ifood
📊 Análise de Dados de Clientes – Comportamento de Consumo
📌 Sobre o Projeto

Este projeto tem como objetivo analisar o comportamento de clientes de uma empresa do segmento de delivery / varejo alimentar, utilizando dados relacionados a perfil demográfico, histórico de compras e interação com campanhas de marketing.

A análise busca identificar padrões de consumo, perfis de clientes mais engajados e oportunidades para melhoria de campanhas e estratégias comerciais.

🎯 Objetivos da Análise ----------------------------------------------------------------

Entender o perfil dos clientes (idade, renda, estado civil e escolaridade)

Avaliar o comportamento de compra por categoria de produto

Identificar clientes com maior potencial de conversão em campanhas

Analisar frequência de compras e canais utilizados

Explorar indicadores que impactam o faturamento e retenção

🗂️ Dataset ----------------------------------------------------------------

O dataset contém aproximadamente 2200 registros e 40+ variáveis, incluindo:

Income → Renda do cliente

Age → Idade

Customer_Days → Tempo como cliente

Kidhome / Teenhome → Quantidade de filhos

Recency → Dias desde a última compra

MntWines / MntMeatProducts / MntFishProducts etc. → Gastos por categoria

NumWebPurchases / NumStorePurchases / NumCatalogPurchases → Canal de compra

NumWebVisitsMonth → Visitas ao site

Response / AcceptedCmp → Resposta a campanhas de marketing

MntTotal → Gasto total do cliente

⚙️ Etapas Realizadas----------------------------------------------------------------

Importação e exploração inicial dos dados

Tratamento de valores nulos e colunas irrelevantes

Conversão de formatos (ex: renda em texto → numérico)

Criação de métricas agregadas (ex: gasto total)

Análise exploratória (EDA)

Identificação de padrões de consumo e comportamento

📈 Possíveis Insights ----------------------------------------------------------------

Clientes com maior renda tendem a apresentar maior gasto total

Compras em canais digitais possuem maior frequência

Determinadas categorias (ex: vinhos e carnes) concentram maior faturamento

Clientes que responderam campanhas apresentam maior recorrência

Tempo como cliente pode impactar o ticket médio
